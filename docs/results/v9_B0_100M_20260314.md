## Key Observations

### Convergence dynamics
- **10-20M:** Strategy unconverged, massive exploitability (123-130 bb/100). Gauntlet deeply negative.
- **30M:** Sharp convergence — exploitability drops to 1.38, gauntlet flips to +471. This is the phase transition.
- **30-100M:** Exploitability stable (1.25-1.38 range). Gauntlet performance continues improving through 100M.
- **Best exploitability:** 90M checkpoint (1.2544)
- **Best gauntlet:** 100M checkpoint (+677.2)

### What 2x schedule + old dampening fixed (vs v7)
1. **CallStationBot** (-437 → +1071): The thin-value leak identified in v9 diagnosis is resolved. The 2x flop/turn schedule gives the solver more iterations to learn value-betting patterns on these streets.
2. **NitBot** (-409 → +719): Aggression properly calibrated. The old all-in dampening (rc<2, 0.7x) controls jam frequency better than the new dampening.
3. **PerturbBot** (-366 → +697): Robustness to near-random noise improved substantially with better convergence on flop/turn.
4. **WeirdSizingBot** (-952 → +220): Still the weakest matchup but now positive. The Phase 3A action grid expansion (3 new sizes) should improve this further.
5. **DonkBot** (+294 → +1024): Large improvement, likely from better flop/turn convergence handling donk-bet sequences.

### Strategy character (100M)
- **Flop OOP:** Heavy donk-small usage (50-72% for EQ5-7), moderate check frequency (20-35% for strong hands). EQ3 checks 99-100% (slowplay or weak mid-range).
- **Preflop IP vs raise:** Very polarized — EQ0-5 fold/call 99%+, EQ7 aggressively 3-bet/all-in.
- **Preflop OOP:** EQ4 limps 100% (consistent with pure check/call at boundary equity).
